# User Story: US-PWA-17c - ประสบการณ์แบบ Native App

**Status:** 🏗 Planned (วางแผน)
**Epic:** [Progressive Web App (PWA)](../01-product-backlog.md#14-progressive-web-app-pwa--ติดตั้งใช้งานแบบ-native-app-บนมือถือ)
**Owner:** UI/UX Dev
**Version:** 1.0 | **Last Updated:** 2026-06-18

---

## 📖 Description
**ในฐานะ** นักศึกษา  
**ฉันต้องการ** ให้แอปพลิเคชันมีการทำงานที่ลื่นไหล แถบหัวขอบสีกลมกลืน หน้าจอตอนเริ่มเปิดตัว (Splash Screen) และไอคอนที่ปรับเข้ากับมือถือ  
**เพื่อให้** รู้สึกถึงประสบการณ์ใช้งานเหมือนติดตั้งแอปพลิเคชันจริงจาก App Store ไม่ใช่แค่พินหน้าเว็บไซต์ธรรมดา  

---

## ✅ Acceptance Criteria
1. [ ] กำหนดค่าไอคอนใน `manifest.json` ให้มีครบทุกขนาดมาตรฐาน:
   * `192x192` (สำหรับหน้าไอคอนทั่วไปและแจ้งเตือน)
   * `512x512` (สำหรับ Splash Screen และไอคอนคุณภาพสูง)
2. [ ] มีไฟล์รูปไอคอนประเภท `maskable` (ขนาด 512x512) โดยให้ขอบนอกมีระยะปลอดภัย (Safe zone) อย่างน้อย 20% เพื่อให้ระบบปฏิบัติการเช่น Android สามารถตัดทรงไอคอนเป็นวงกลม สี่เหลี่ยมมุมโค้ง หรือสี่เหลี่ยมตามธีมระบบได้โดยรูปภาพสำคัญไม่ตกขอบ
3. [ ] กำหนดตัวแปร `theme_color` และ `background_color` ให้ตรงกับ Design System หลัก (โทนสีสว่าง/มืดตามลักษณะแอป) ทำให้ขอบระบบบราวเซอร์ซิงก์เป็นสีโทนเดียวกัน
4. [ ] ตัวแอปเปิดตัวในโหมดแนวตั้งเสมอตลอดการรัน (`orientation: "portrait"`) เพื่อรักษาโครงสร้างหน้าจอ UI และการจัดวางข้อมูลกิจกรรมไม่ให้บิดเบี้ยว
5. [ ] เมื่อกดเปิดใช้งาน หน้าจอเริ่มแรก (Splash Screen) บนสมาร์ทโฟนจะดึงไอคอนขนาด 512x512 มาแสดงตรงกลางหน้าจอพร้อมพื้นหลังสีเดียวกันได้อย่างสวยงาม

---

## 🛠 Technical Tasks (งานพัฒนาที่ต้องทำ)
- [ ] จัดเตรียมและตัดแบ่งภาพโลโก้ `public/smocamt-logo-icon.png` ให้เป็น:
  * `icon-192x192.png`
  * `icon-512x512.png`
  * `icon-512x512-maskable.png` (แก้ไขให้รูปตราสัญลักษณ์มีระยะห่างจากขอบอย่างน้อย 10% หรือ 20% เพื่อกันการโดนตัดแต่งขอบระบบ)
- [ ] ทำการอ้างอิงภาพไอคอนทั้งหมดในส่วนของ `icons` อาร์เรย์ใน `public/manifest.json`
- [ ] ใส่รหัส `<meta name="theme-color" content="#COLOR_HEX">` ในไฟล์ `src/app/layout.tsx` และตั้งค่าใน Manifest เพื่อรองรับการเปลี่ยนสีขอบหน้าจอตามธีม
- [ ] ทดสอบและเปรียบเทียบพฤติกรรมการเรียกแสดงหน้าจอเปิดตัวแอป (Splash Screen) บนอุปกรณ์ Android หรือ Android Emulator
