## ฝั่งนักศึกษา (สิ่งที่ user จะเห็น)

- **ลงทะเบียนกิจกรรมตามเวลาเปิด–ปิด** — แต่ละกิจกรรมกำหนดช่วงเวลาเปิดรับ–ปิดรับได้แล้ว ปุ่มลงทะเบียนจะโผล่/หายตามเวลาจริง
- **รองรับ Walk-in** — ถึงที่นั่งลงทะเบียนล่วงหน้าจะเต็ม ก็ยังเดินมาเช็คอินหน้างานได้ (มีโควต้า Walk-in แยกบวกเพิ่มจากโควต้าหลัก)
- **โปสเตอร์กิจกรรมหลายรูป** — ปัดดูได้หลายรูป แล้วก็ซูมเต็มจอได้ด้วย pinch / แตะสองที / scroll เมาส์
- **แบบฟอร์ม KAS** — มีฟอร์มก่อน/หลังกิจกรรมแบบ K, A, S หน้าตาประมาณ Google Forms มีให้คะแนนแบบ quiz และมีเงื่อนไขข้ามข้อได้
- **Leaderboard รายบุคคล** — ดูอันดับตัวเองได้ง่ายขึ้น เห็นคะแนนกับอันดับของตัวเองทันที (ถ้าคะแนน 0 จะขึ้นว่า "ยังไม่ถูกจัดอันดับ")
- **ประวัติกิจกรรมแม่นขึ้น** — โชว์ลำดับเช็คอินจริง (เช่น เช็คอินเป็นคนที่ #5 จาก 400) และต้องเช็คอินจริงก่อนถึงจะส่งแบบประเมินได้
- **รองรับ 4 ภาษา** (ไทย/อังกฤษ/พม่า/จีน) ฟอนต์ไทยคมขึ้น โหลดเร็วขึ้น

## ฝั่งทีม (technical changelog)

### Security
- QR token เซ็นด้วย HMAC-SHA256 อายุ 5 นาที เช็คแบบ timing-safe
- Audit log แบบกันแก้ย้อนหลัง — ร้อยกันเป็น hash chain (SHA256) + advisory lock ตอนเขียน และให้คะแนนแบบ atomic
- ตามเก็บ audit findings #2–15 + quick wins, ใส่ security headers/CSP, harden ส่วน upload, ลดการเก็บ PII เท่าที่ไม่จำเป็น
- ปรับ session อายุ 7 วัน และอัป dependencies
- ขยายขอบเขตที่ลง log: แก้/เปลี่ยน role/ลบ user, แก้ฟอร์ม, admin login + แก้เคส chain ขาด
- หน้า Audit Logs ทำ pagination ฝั่ง server 30 รายการ/หน้า + ใส่ index ที่ `timestamp`

### ระบบฟอร์ม KAS
- หนึ่งกิจกรรมมีได้หลายฟอร์ม: `K_pre` / `K_post` / `A` / `S`
- แบ่ง section แบบ Google Forms, branching ตามคำตอบ, ซ่อน/แสดงข้อแบบมีเงื่อนไข, และให้คะแนนแบบ quiz
- ปิดฟอร์มและมอบคะแนนอัตโนมัติตามเวลา (`opensAt` / `closesAt`, `isAwarded`)
- export คำตอบ/คะแนน/เวลา เป็น XLSX

### Events & Quota
- เพิ่มช่วงเวลาเปิด–ปิดลงทะเบียน (registration open/close) ปุ่มลงทะเบียนทำงานตามเวลานี้
- โควต้า Walk-in แยกบวกเพิ่มจากโควต้าหลัก + มีเพดานของ Walk-in เอง (บังคับแบบ atomic กัน race condition)
- โควต้าแยกกลุ่มไทย/อินเตอร์ + อัปไอคอน scanner กับคำแปล
- รองรับโปสเตอร์หลายรูปต่อกิจกรรม (`imageUrls`) + carousel ปัดได้ + ซูมเต็มจอ

### Leaderboard
- คืนอันดับรายบุคคลครบทุกคน (ไม่ตัดแค่ Top 50) และโชว์อันดับ/คะแนนตัวเองได้ชัวร์บน production
- คนที่ 0 คะแนนขึ้นเป็น "ยังไม่ถูกจัดอันดับ" (Unranked)

### DB & Stability
- เปลี่ยน timestamp เป็น `timestamptz` และตั้ง timezone ของ DB เป็น Asia/Bangkok
- ใส่ `ON DELETE` ให้ FK ที่ชี้มาที่ user + เคลียร์แถวที่เกี่ยวข้องก่อนลบ user
- แก้ connection pool starvation, ใส่ timeout ให้ route กันค้างยาว, แก้ปัญหา polling ซ้อนกัน
- เรียงประวัติด้วยลำดับเช็คอินจริง (ไม่ใช่ลำดับลงทะเบียน)

### UI/UX & Performance
- แก้คำที่ตัดไม่สวยตอนชื่อยาว (การ์ดประวัติ, ปุ่ม CTA ฟอร์ม, chip เลือกฟอร์ม)
- แก้หน้า login ให้ขึ้น error สวยขึ้น + เปิด `trustHost` ให้ OAuth/PKCE
- refresh JWT ทันทีหลังกรอกโปรไฟล์เสร็จ (ไม่ต้อง log out)
- ลดขนาดฟอนต์/รูป/bundle ปรับ FCP/LCP/CLS และแก้ฟอนต์ไทยให้ render คมขึ้น
- ไล่เก็บ `any` เปลี่ยนเป็น type ที่ปลอดภัยขึ้น
